学习内容

1.标签

  • 1.图片标签

    标签为img

    • img的属性：
    • src：图片的网址
    • alt：图片的描述
    • width：图片的宽度

    表现形式为一对"<>"中写“img src="图片的网址" alt="图片的描述" width="图片的宽度"”

  • 2.超链接标签

    GitHub

    只是打个比方，别真点

    标签为a

    • a的属性：
    • href：超链接的网址

    表现形式为一对"<>"中写“a href="超链接的网址", 关闭为"<>"中写"/a"

  • 3. 文字强调

    这是加粗的文字。

    这是斜体的文字。

    文字加粗是strong标签，文字斜体是em标签。

    加粗表现形式为“<>”中写"strong“，关闭为"<>"中写"/strong"

    斜体表现形式为“<>”中写"em“，关闭为"<>"中写"/em"

  • 4. 引用内容

    编程不是关于你知道什么，而是关于你能用你知道的东西做什么。

    标签为blockquote

    表现形式为一对"<>"中写"blockquote,"，关闭为"<>"中写"/blockquote"

    blockquote标签用于将一段内容进行引用，并显示为斜体。

  • 5. 分隔线

    上面的内容和下面的内容被一条线分隔

    这是上面的内容

    这是下面的内容

    hr标签用于将内容分隔，并显示为水平线。

    hr标签没有属性。

    表现形式为“<>"中写"hr"，关闭为"<>"中写"/hr"
